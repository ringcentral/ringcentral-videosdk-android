Detailed Description

The VideoController class is a managing class to control the video-related functions and states in an active meeting, such as mute or unmute local or remote video.

pure virtual

Stops subscribing to the video stream of a remote user.

A successful call of muteRemoteVideoStream triggers the onlocalVideoMuteChanged event callback to the specified remote user, other meeting users will retrieve the onRemoteVideoMuteChanged event notification.

Note
Only the meeting host or moderator has permission to invoke this method.
Parameters
uidThe unique id of the specified remote user
Returns
